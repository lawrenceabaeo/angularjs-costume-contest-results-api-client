AngularJS web app that displays costume contest results retrieved from a REST API

What is this?

This mini-project displays a list of costume contests. Links from those contests display a list of entries and winner results. It is created using the AngularJS front end client framework. It consumes a REST API from another mini-project ( click here to see the REST project )

Click here to see the AngularJS Costume Contests Results app!

Features:

  • AngularJS routing
  • Mobile responsive (uses bootstrap grid system)

Why did I make this?

This example project was an effort to learn AngularJS, and how an AngularJS app can hook into an external API.

What technology skills were learned/retreived?

  • AngularJS
  • Bootstrap
  • Updated the Rails server app for CORS
